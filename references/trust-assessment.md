# 信任建立评估 · Simulator 复盘 Reference

> 用途:本次模拟挂载本框架时,Phase 3 专项段按此结构给参考反馈(不打分)。
> 谁读:Agent。给学员的反馈遵循措辞规则(对学员侧框架名显示为"信任建立";不出现"信任方程式 / Maister / 可信度 / 可靠度 / 亲近度 / 自我导向"等理论词)。
> 适用场景:关系类 —— 信任建立 / 高管关系经营 / 关系破冰(首次或早期建立信任、长期关系经营、被当成"又一个供应商"时)。
> 来源:David Maister, Charles Green, Robert Galford《The Trusted Advisor》信任方程式 T = (C + R + I) / S。本结构为该公式在 ToB 销售复盘场景的改编版。内容经改写以符合授权要求。
> 配套:本 reference 是"复盘专项段"用的精简检查表;完整的信任五框架(关系阶梯 / Lewicki-Bunker / Speed of Trust / Maister 五步)见 `trust-frameworks.md`,两份配合使用。

## 一、框架是什么(一句话)

信任不是"喜不喜欢你",是四件事算出来的:他信你**懂行**、信你**说到做到**、愿意对你**说真话**,再减去他觉得你**有多自私**。前三个相加、第四个做除数 —— 意思是只要他觉得你满脑子是自己的单子,前面三项做得再好也会被打很大折扣。

> 公式(给 Agent,**不对学员写公式**):T = (C + R + I) / S

---

## 二、"步骤"结构(复盘按此顺序)

> 跟关键对话 / 谈判不同:这四项**不是先后步骤,是四个并列变量**。复盘时按下面顺序逐项看,但**不要求**"前一项做到才能做下一项"。第 4 项是除数,单独最关键。

| 顺序 | 对学员表述(给学员看) | 这一项要做到 | 公式变量(给 Agent,不对学员说) |
|---|---|---|---|
| 1 | 让他相信你是懂行的 | 用专业、具体案例、对他业务的理解,让他觉得"这人内行" | C · Credibility 可信度 |
| 2 | 让他相信你说到做到 | 承诺的事记住并兑现;不夸口、不放空炮 | R · Reliability 可靠度 |
| 3 | 让他愿意对你说点真话 | 营造安全感,让他敢说非官方的、内部的实情 | I · Intimacy 亲近度 |
| 4 | 别让他觉得你只想着自己的单子 | 把对话重心放在他的问题上,而不是"我要卖" | S · Self-Orientation 自我导向(除数) |

**关于第 4 项(给 Agent)**:S 是分母,杀伤力最大。一场对话里 C / R / I 都不错但 S 高,整体信任仍会被除小。复盘时如果只能让学员改一件事,**优先压 S**。

## 三、怎么从对话里识别"做到 / 没做到"

> 每条反馈带模拟里第 N 轮证据。

### 第 1 项 · 让他相信你是懂行的(C)
- ✅ 做到:讲可核对的具体案例 / 数字;听得懂他的业务术语并接得上;问到点子上的专业问题。
- ❌ 没做到:用"客户都很满意"这种空话;讲不出具体数字;对他行业 / 业务一知半解被问住。

### 第 2 项 · 让他相信你说到做到(R)
- ✅ 做到:记得并回应上次承诺的事;这次答应的事说清楚什么时候交;不夸海口。
- ❌ 没做到:忘了上次说过什么;给"绝对没问题 / 保证"这种兜不住的承诺;答应的事含糊不清。

### 第 3 项 · 让他愿意对你说点真话(I)
- ✅ 做到:他主动说了非官方的内部情况 / 顾虑 / 私下看法;愿意聊业务之外的话题;回应变长、变主动。
- ❌ 没做到:全程官方说法;只被动回应、惜字如金;你急着讲方案没给他开口的空间。

### 第 4 项 · 别让他觉得你只想着自己的单子(S)
- ✅ 做到:开场先问他的问题再谈方案;说话重心是"你需要什么"而不是"我们多强";不急着逼承诺。
- ❌ 没做到:开场就推方案 / 讲公司;说话长度远超客户、抢话;反复往成交上带、逼他当场表态。

> **底层判断依据(Agent 参考,不对学员复述术语)**:
> - **四变量定位**:客户行为异常时反推缺哪块 —— 不回邮件→C / I 弱;答应却不执行→S 高;见你但不讲真话→I 低;愿意讲但不签→C 弱。
> - **关系档位修正**:同一行为在不同关系档位意义不同,配合 `trust-frameworks.md` 的关系阶梯一起读。
> - **S 是除数**:优先级永远先压 S,再补 C / R / I。

## 四、专项段措辞示范(给学员看的样子)

> 按四项顺序;每项下可有多条,各带信号灯;带第 N 轮证据;[可改进] / [踩雷] 带"下次怎么做"。

```
## 专项复盘 · 信任建立

> 这是本次「信任建立」专项训练的表现反馈。下面按建立信任的四个方面,逐项看你这次做得怎么样。

### 第 1 项 · 让他相信你是懂行的
- [可改进] 第 6 轮你说"客户都很满意",他第 7 轮立刻追问"具体哪些客户"—— 空话反而让他起疑。
  下次:换成"过去 5 年做过 X 个,其中 Y 个交付情况是这样(具体数字)"。

### 第 2 项 · 让他相信你说到做到
- [未涉及] 这次是二次见面,还没机会兑现承诺。下次见面前,把上次答应他的事主动做掉一件,开场就提。

### 第 3 项 · 让他愿意对你说点真话
- [可改进] 整场 12 轮他只主动开口 1 次,大部分惜字如金 —— 他还没对你打开。
  下次:前 2-3 轮别讲方案,先问他业务上的具体难处,给他开口的空间。

### 第 4 项 · 别让他觉得你只想着自己的单子
- [踩雷] 前 4 轮你说话长度是他的 3 倍,全在讲"我们能做什么"—— 他会觉得你急着卖。
  怎么补救:开场重心挪到"他的问题"上,等他主动问"你们能做什么"再切方案。

### 这次最值得带走的
你这次信任没往上走,主要不是能力问题,是顺序问题 —— 太早、太多讲自己。把开场重心从"介绍我们"挪到"了解他",信任就会开始动。
```

## 五、术语对照(原名 → 对学员表述)

| 理论 / 原著说法(不对学员出现) | 对学员说 |
|---|---|
| 信任方程式 / Trust Equation / Maister | 建立信任的几个方面 |
| Credibility / 可信度 | 让他相信你是懂行的 |
| Reliability / 可靠度 | 让他相信你说到做到 |
| Intimacy / 亲近度 | 让他愿意对你说点真话 |
| Self-Orientation / 自我导向 | 他觉不觉得你只想着自己的单子 |
| 分母 / 除数 | 会把前面都打折的那一项 |
| 关系阶梯 / 信任档位 | 你在他眼中是什么角色 |

> 框架名"信任建立"可对学员出现(受控例外)。上述机制类 / 理论类术语一律翻成右栏说法。
